## 4. 자동 패치 스크립트 (선택)

`scripts/patch_docx.py` 가 생성되어 있음 (다음 commit). 실행:
```bash
python scripts/patch_docx.py \
    --input thesis_current/박사논문_1장_서론_v4.docx \
    --output thesis_current/박사논문_1장_서론_v5.docx \
    --patch ch1
```

⚠️ **수동 Word 적용 권장**: 한국어 docx 서식 보존이 중요하므로 Word의 찾기-바꾸기 기능으로 위 변경을 하나씩 적용하는 것이 가장 안전하다. 자동 스크립트는 편의 도구일 뿐 검증 필수.
