# 📖 박사논문 용어 정리 (v5 재작성 기준)

> 모든 장에서 **동일 용어를 동일 의미로** 일관되게 사용하기 위한 사전.
> 작성: 2026-04-21. 새 실측 수치 반영. 논문 전체의 기준 문서.

## 1. 프레임워크 이름

| 용어 | 정의 | 영문 |
|---|---|---|
| **Triple-Hybrid RAG** | Vector, Graph, Ontology 3가지 지식 소스를 가중치 융합하는 RAG 아키텍처 (본 연구 및 선행 연구 공통) | Triple-Hybrid Retrieval-Augmented Generation |
| **R-DWA** | Rule-based Dynamic Weighting Algorithm. 선행 논문(JKSCI 2025)의 규칙 기반 가중치 결정 방식. 2단계 (질의 유형 → 기본 가중치, 밀도 신호 → 연속 조정) | Rule-based Dynamic Weighting Algorithm |
| **L-DWA** | **Learned** Dynamic Weighting Algorithm. 본 박사논문의 새 기여. PPO로 학습된 Actor-Critic 정책이 상태(18-dim)를 보고 가중치(α, β, γ)를 출력 | Learned Dynamic Weighting Algorithm |
| **Oracle** | **상한선 정책**. 각 질의에 대해 cache 안의 66개 가중치 중 total_reward 최고값을 선택. 실제 배포 불가능 (정답을 알아야 함). L-DWA의 개선 여지를 정량화하는 기준 | Oracle policy (upper bound) |

## 2. 지식 소스 (Source)

| 용어 | 정의 | 가중치 기호 |
|---|---|---|
| **Vector 소스** | 문서 임베딩 기반 FAISS 검색. 의미 유사도 상위 k개 문서 반환 | **α** |
| **Graph 소스** | NetworkX로 구축된 지식 그래프 상 BFS (3-hop). 개체-관계 경로 반환 | **β** |
| **Ontology 소스** | Owlready2 기반 OWL 온톨로지 + 규칙 fallback. 클래스 계층 + 제약 조건 만족 엔티티 반환 | **γ** |
| **Weight simplex** | (α, β, γ) ∈ Δ³, 즉 α + β + γ = 1, α,β,γ ≥ 0 | — |

## 3. 평가 지표

| 용어 | 정의 | 목적 |
|---|---|---|
| **F1_strict** | 한국어 조사 제거 + 구두점 제거 + NFC 정규화 후 **토큰 집합** 기반 F1. 본 논문의 **엄격한** 주 지표. | 재현 가능 + 엄격 |
| **F1_substring** | Gold를 쉼표로 분할하여 얻은 목록 항목들이 prediction에 부분문자열로 나타나는지 검사. **선행 논문(JKSCI 2025)이 사용한 것으로 추정**되는 loose 평가와 호환. | Journal 비교 |
| **EM (Exact Match)** | 정규화된 prediction과 gold의 완전 일치. 본 벤치마크에서 항상 0 (gold가 콤마 리스트 형식) → **본 논문에서 주 지표 아님** | 참고용 |
| **Faithfulness** | RAGAS 근사. Prediction의 각 문장이 retrieved context에 의해 지지되는 비율 | 답변 근거 |
| **Latency** | End-to-end 응답 시간 (초). 5초 초과 시 reward에 penalty | 실용성 |
| **Total Reward R** | `R = 0.5·F1_strict + 0.3·EM + 0.2·Faithfulness − 0.1·max(0, latency−5.0)`. PPO 학습의 목적 함수. | PPO 학습 목표 |

## 4. 질의 분류 체계

| 용어 | 정의 | 예시 | 비중 (gold_qa_5000) |
|---|---|---|---|
| **Simple** | 단일 속성 또는 관계 질의 | "김철수 교수의 소속 학과는?" | 2,000 (40%) |
| **Multi-hop** | 2개 이상의 관계를 연속 탐색해야 하는 질의 | "이영희 교수가 담당하는 과목을 수강하는 학과는?" | 1,750 (35%) |
| **Conditional** | 수치/범위/논리 제약을 포함하는 질의 (본 논문의 boundary query에 해당) | "40세 이하 컴공과 교수 중 AI 연구자는?" | 1,250 (25%) |
| ~~Boundary~~ | 구 용어 (v2). "Conditional"과 사실상 동일. **v5에서 Conditional로 통일**. |

## 5. PPO/RL 용어

| 용어 | 정의 |
|---|---|
| **MDP** | Markov Decision Process. 본 논문에서 Triple-Hybrid 가중치 결정을 1-step contextual bandit으로 공식화 |
| **State (18-dim)** | `[density(3) + intent_logits(3) + source_stats(9) + query_meta(3)]` (Ch.5 Eq.5-1) |
| **Action** | (α, β, γ) ∈ simplex. Dirichlet 분포에서 샘플링 |
| **Actor-Critic** | 공유 backbone (18→64→64 Tanh) + Actor head (Dirichlet 농도) + Critic head (가치). **5,636 파라미터** |
| **PPO** | Proximal Policy Optimization. Clip ratio 0.2, epochs 4, mini-batch 8 |
| **Rollout** | Episode당 32개 (질의, 가중치, 보상) 샘플 수집 |
| **Episode** | 1 rollout → policy update 사이클. 본 학습은 seed당 **10,000 episodes** |
| **Seed** | 난수 시드. 본 논문은 {42, 123, 999} 3개 seed로 mean ± std 보고 |

## 6. 엔지니어링 용어

| 용어 | 정의 |
|---|---|
| **Offline reward cache** | 5,000 질의 × 66 이산 가중치 = **330,000 엔트리**의 reward 사전 계산 SQLite DB. PPO 학습 중 API 호출 0. |
| **Discretization grid** | 가중치를 0.1 단위로 이산화. 0~10 정수 그리드, 합이 10인 삼각형 점 = 66개 |
| **Retrieval cache** | `evaluate_rerun.py`의 per-query retrieval 결과 pickle. 정책 여러 번 재측정 시 시간 절약 |
| **State cache** | `train_ppo.py`의 per-query state pickle. 여러 seed 학습 시 재사용 |
| **Snap-to-simplex** | 연속 Dirichlet 샘플을 정수 simplex (66 점 중 1)에 매핑 → cache 룩업 가능 |

## 8. 선행 논문 / 본 논문 구분

| 구분 | 선행 논문 (JKSCI 2025) | 본 박사 논문 |
|---|---|---|
| 저자 | 신동욱, 문남미 | 신동욱 (문남미 지도) |
| 기여 | Triple-Hybrid RAG 아키텍처 제안, R-DWA | **L-DWA (PPO 기반)** + Offline cache + 이중 F1 평가 체계 + Cross-domain 분석 |
| 보고 수치 | F1 0.86, EM 0.78, Faith 0.89 (선행의 자체 평가) | R-DWA F1_strict 0.137 / F1_substring 0.448, L-DWA F1_strict 0.167 / F1_sub 0.488 (본 논문의 공정 재평가) |

## 9. v4 → v5 **용어 변경 (중요)**

| v4 사용 | v5 사용 | 이유 |
|---|---|---|
| "Boundary query" | "Conditional query" | 데이터셋의 공식 유형 명칭이 conditional. Boundary는 해석 용어로 혼동 |
| "F1 0.89" (단일) | "F1_strict 0.167 / F1_substring 0.488" (병기) | 평가 기준 투명성 |
| "Boundary EM +32.8%" | "Conditional F1_substring +131%" | EM=0 구조적 한계로 F1_substring으로 대체 |
| "Zero-shot 도메인 이전" | "Domain-specific learning" | 실험적으로 zero-shot transfer 실패 |
| "F1 +3.5%" | "F1_strict +21.9% / F1_substring +8.4%" | 실측 반영 |

## 10. 수식 기호

| 기호 | 의미 |
|---|---|
| α, β, γ | Vector, Graph, Ontology 가중치 (simplex) |
| s_e, s_r, s_c | Entity, Relation, Constraint density (RuleBasedIntent 출력) |
| s_t, a_t, r_t | 상태, 행동, 보상 (PPO) |
| π_θ | 정책 (Actor network) |
| V_φ | 가치 함수 (Critic network) |
| γ_PPO | PPO 할인 계수 (0.99, 본 논문에서는 1-step이라 실효 영향 없음) |
| λ_GAE | GAE lambda (0.95) |
| ε_clip | PPO clip ratio (0.2) |
| H | Policy entropy |
| λ_R (R-DWA) | R-DWA의 밀도 조정 강도 (0.3) |

## 사용 가이드

- 본 문서는 **정의의 단일 출처** (single source of truth)
- 각 장 초안 작성 시 이 문서의 용어/수치만 인용
- v4 용어/수치가 남아 있으면 심사 시 **내용적 모순**으로 지적당할 수 있으므로 v5에서 완전 통일
