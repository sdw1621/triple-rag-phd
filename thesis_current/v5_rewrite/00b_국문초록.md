# 국문초록

검색 증강 생성(RAG)은 대형 언어 모델(LLM)의 환각과 지식 단절 문제를 완화하는 핵심 기법이며, 최근 단일 지식 소스의 한계를 극복하기 위해 Vector, Graph, Ontology 세 소스를 결합한 Triple-Hybrid RAG 아키텍처가 제안되었다. 본 연구의 선행 논문 Shin & Moon (2025, JKSCI) 는 Rule-based Dynamic Weighting Algorithm (R-DWA) 으로 세 소스의 가중치 (α, β, γ) 를 2단계 규칙으로 결정하였으나, 규칙 기반의 구조적 한계 (도메인 편향, 단일 λ, 조건부 질의 미세 분해 부재, 도메인 이전 불가) 를 가진다.

본 박사 논문은 **3단계 기여** 로 이 한계를 해소한다.

**(1) 재현성 기여.** 선행 JKSCI 논문의 재현성 장애 3가지 — (i) `evaluator.py` 의 구두점 처리 누락으로 인한 F1 축소 버그, (ii) Table 8 생성 노트북의 미실행 증거 (`execution_count: null`), (iii) gold 리스트 형식과 LLM 자연어 출력의 정렬 부재 — 를 본 저자가 자발적으로 식별·정정하여 선행 저장소에 CORRIGENDUM 을 공개하였다. 이로써 **신뢰 가능한 새 기준점 (corrected baseline: S1+S2+S3 모두 적용된 R-DWA, F1<sub>strict</sub> 0.529)** 을 수립하였다.

**(2) 방법론 기여.** Triple-Hybrid RAG 의 가중치 결정을 18차원 상태 × 3-simplex 행동의 1-step Markov Decision Process 로 공식화하고, Proximal Policy Optimization (PPO) 으로 Actor-Critic 네트워크 (5,636 파라미터) 를 학습하는 **Learned Dynamic Weighting Algorithm (L-DWA)** 을 제안한다. 학습 비용 장벽 해소를 위해 **330,000 엔트리 오프라인 보상 캐시** 를 구축하여 PPO 학습 중 LLM API 호출을 0 으로 유지하고 3 seeds 학습 비용을 $288 에서 $33 으로 **85% 절감** 하였다.

**(3) 실증 기여.** Corrected baseline 위에서 5,000 QA × 3 seeds 실험 결과, L-DWA 는 R-DWA 대비 F1<sub>strict</sub> 0.529→**0.562 ± 0.007 (+6.2%)**, F1<sub>substring</sub> 0.482→**0.507 ± 0.009 (+5.2%)**, Faithfulness 0.544→**0.580 ± 0.009 (+6.6%)** 의 일관된 개선을 기록하였다. 특히 L-DWA 의 Dirichlet 평균이 **per-query argmax Oracle (66-점 이산 격자 상한) 을 모든 F1 지표에서 초과** 한다 (0.562 > 0.554) — 학습된 연속 정책이 이산 argmax 의 상한을 넘어서는 **연속 공간 탐색력** 을 실증한다. 조건부 질의 유형에서는 F1<sub>strict</sub> **+36.7%** (0.223→0.304) 로 Oracle 을 초과하여 per-query 학습형 가중치의 질적 우월성을 확립한다.

**교차 도메인 분석.** HotpotQA/MuSiQue/PubMedQA 에서의 naive transfer 실패 (−30~35%) 의 원인을 L1 언어 장벽·L2 도메인 어휘·L3 Graph/Ontology 부재의 **3-층 구조** 로 분해하였다. 영어 intent 패턴 추가 (L1 해소) 시 R-DWA **+38%**, L-DWA **+49%** 회복을 실측하였으며, **영어 합성 대학 벤치마크** (403 QA) 구축 실험에서 R-DWA 가 F1<sub>strict</sub> **0.663** (Korean 0.529 대비 **+25%**) 을 달성하여 **L-DWA 아키텍처가 언어 중립적** 임을 실증하였다. 본 논문은 이로써 cross-domain 실패를 **아키텍처 한계** 에서 **데이터 alignment 문제** 로 재포지셔닝한다.
