# 국문초록

검색 증강 생성(RAG)은 대형 언어 모델(LLM)의 환각과 지식 단절 문제를 완화하는 핵심 기법이며, 최근 단일 지식 소스의 한계를 극복하기 위해 Vector, Graph, Ontology 세 소스를 결합한 Triple-Hybrid RAG 아키텍처가 제안되었다. 기존 연구인 Shin & Moon (2025, JKSCI)의 Rule-based Dynamic Weighting Algorithm (R-DWA)은 질의 유형별 기본 가중치와 밀도 신호 기반 연속 조정의 2단계 규칙으로 세 소스의 가중치 (α, β, γ)를 결정하나, (i) 기본 가중치 테이블의 도메인 편향, (ii) 단일 하이퍼파라미터 λ의 한계, (iii) 조건부 질의 유형의 세부 미구분, (iv) 도메인 이전 불가의 네 가지 구조적 한계를 가진다.

본 논문은 Triple-Hybrid RAG의 가중치 결정을 18차원 상태 × 3-simplex 행동의 1-step Markov Decision Process로 공식화하고, Proximal Policy Optimization (PPO)로 Actor-Critic 네트워크(5,636 파라미터)를 학습하는 **Learned Dynamic Weighting Algorithm (L-DWA)**을 제안한다. 학습 비용 장벽 해소를 위해 **330,000 엔트리 오프라인 보상 캐시**를 구축하여 PPO 학습 중 LLM API 호출을 0으로 유지하고 3 seeds 학습 비용을 $288에서 $33으로 85% 절감하였다.

합성 대학 행정 5,000 QA 벤치마크에서 3 seeds (42, 123, 999) 실험 결과, L-DWA는 R-DWA 대비 F1_strict 0.137→0.167 **+21.9%**, F1_substring 0.450→0.488 **+8.4%**, Faithfulness 0.835→0.866 **+3.6%**, Total Reward 0.199→0.215 **+7.6%**의 통계적으로 유의미한 개선을 달성하였다. 특히 **조건부 질의 유형**(n=1,250)에서 F1_substring 0.090→0.208의 **+131% 상대 개선**을 기록하였고, 이는 per-query argmax Oracle의 0.195를 상회하여 per-query 학습형 가중치의 질적 우월성을 실증한다. L-DWA는 Oracle 상한 성능의 **F1_strict 99.4%, Faithfulness 97.5%**에 도달하며, **F1_substring에서는 Oracle을 초과**한다.

또한 본 논문은 **이중 F1 평가 체계** (F1_strict: 한국어 조사·구두점 제거 기반 엄격, F1_substring: 콤마 리스트 gold 호환 loose)를 도입하여 선행 연구와의 호환성과 엄격 재평가의 투명성을 동시에 제공한다. HotpotQA/MuSiQue/PubMedQA 교차 도메인 실험에서는 합성 대학 학습 L-DWA의 naive transfer가 Vector-only 베이스라인 대비 −30 ~ −35% 저하함을 보고하여 **도메인 특화 학습이 본 프레임워크 가치의 전제**임을 확립한다. 모든 코드·캐시·체크포인트는 GitHub (sdw1621/triple-rag-phd)에 공개되어 완전 재현 가능하다.
