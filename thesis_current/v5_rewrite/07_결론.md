# 제Ⅶ장 결론 (v5 재작성)

## 1절 연구 요약

본 박사 논문은 선행 JKSCI 2025 논문의 재현성 장애 **3가지 (평가기 구두점 버그, Table 8 노트북 미실행, gold-prompt 형식 불일치)** 를 자발적으로 식별·정정하여 신뢰 가능한 **corrected baseline** 을 수립하고, 그 위에서 Triple-Hybrid RAG 의 가중치 결정을 **Markov Decision Process 로 공식화한 최초의 정식화** 와 **PPO 기반 Learned DWA (L-DWA)** 를 제안하였다.

새 기준점 (S1 구두점 수정 + S2 프롬프트 정렬 + S3 2-branch faith) 위에서 L-DWA 는 R-DWA 대비 **F1<sub>strict</sub> +6.2%, F1<sub>substring</sub> +5.2%, Faithfulness +6.6%** 의 일관된 개선을 기록하며, 특히 **Oracle (per-query argmax, 66-점 이산 격자 상한) 을 모든 F1 지표에서 초과** 하였다 (0.562 > 0.554). 이는 학습된 Dirichlet 평균이 이산 argmax 상한을 넘어서는 **연속 공간 탐색력** 을 실증한다.

선행 논문의 규칙 기반 R-DWA 의 네 가지 구조적 한계 — (i) 도메인 편향된 기본 가중치 테이블, (ii) 단일 λ 하이퍼파라미터, (iii) 유형 내 미세 구분 부재, (iv) 도메인 이전 불가 — 를 학습형 접근으로 해소하고, 합성 대학 5,000 QA 벤치마크에서 정량적 개선을 실증하였다.

## 2절 본 논문의 기여

### 2.1 방법론적 기여

**기여 1 — MDP 공식화.** Triple-Hybrid RAG의 가중치 결정을 18차원 상태 공간과 3-simplex 행동 공간의 1-step contextual bandit MDP로 공식화하였다. 이는 하이브리드 RAG 연구 흐름에서의 **최초 정식화** 이다.

**기여 2 — PPO-based L-DWA.** 공유 backbone (18→64→64 Tanh) + Actor head (Softplus Dirichlet 농도) + Critic head 로 구성된 **5,636 파라미터의 경량 Actor-Critic** 를 PPO (clip ratio 0.2, 10,000 episodes) 로 학습하여 3 seeds 전부 Total Reward 0.215 ± 0.002 의 일관된 수렴을 확보하였다.

### 2.2 엔지니어링 기여

**기여 3 — 오프라인 보상 캐시.** 5,000 질의 × 66 이산 가중치 = **330,000 엔트리의 reward 를 SQLite 에 사전 계산** 하여 PPO 학습 중 LLM API 호출 수를 0 으로 만들었다. 3 seeds 학습의 비용을 $288 → $33 (**−85%**), 실 wall-clock 을 50h+ → 1h PPO (**−98%**) 로 축소하였다.

**기여 4 — 이중 F1 평가 체계.** 한국어 조사·구두점 제거 기반의 엄격한 F1_strict 와 콤마 리스트 gold 에 호환되는 F1_substring 을 병기 보고하여, 선행 연구 (JKSCI 2025) 와의 연속성과 엄격한 재평가의 투명성을 동시에 제공하였다. 본 논문은 두 지표의 차이 자체를 **한국어 QA 평가의 방법론적 관찰** 로 기록한다.

### 2.3 실증적 기여 (Corrected Baseline 위에서)

**기여 5 — L-DWA 성능 정량화 (5,000 QA, 3 seeds, corrected baseline):**

| 지표 | R-DWA baseline | **L-DWA (3-seed 평균)** | Δ | Oracle |
|---|---|---|---|---|
| F1<sub>strict</sub> | 0.529 | **0.562 ± 0.007** | **+6.2%** | 0.554 (초과) |
| F1<sub>substring</sub> | 0.482 | **0.507 ± 0.009** | **+5.2%** | 0.504 (초과) |
| F1<sub>char</sub> | 0.469 | **0.494 ± 0.010** | **+5.3%** | 0.487 (초과) |
| EM | 0.387 | 0.388 ± 0.001 | +0.3% | ≈ Oracle |
| Faithfulness | 0.544 | **0.580 ± 0.009** | **+6.6%** | 0.570 (초과) |

**기여 6 — L-DWA 가 Oracle 상한을 초과.** L-DWA 는 per-query argmax Oracle (66-점 이산 격자) 을 **모든 F1 지표에서 초과** 한다. Dirichlet 평균이 이산 격자 바깥의 연속 공간을 탐색하여 이산 argmax 상한을 넘어선다 — 학습된 연속 정책의 **가장 강력한 학술적 주장**.

**기여 7 — Conditional 질의의 질적 개선.** Conditional 유형 (n=1,250) 에서 L-DWA 는 F1<sub>strict</sub> **0.304** 를 기록하여 R-DWA 의 0.223 대비 **+36.7% 상대 개선** 을 달성하며, Oracle 의 0.290 를 상회한다. 이는 per-query 학습형 가중치의 규칙 기반 대비 **질적 우월성** 에 대한 본 논문의 가장 강력한 per-type 증거이다.

**기여 8 — 교차 도메인 실패의 원인 분해 + 실증 (Ch.6 §7).** 대학 학습 L-DWA 의 HotpotQA/MuSiQue/PubMedQA naive transfer 실패를 세 층위로 분해하였다:

- **L1 언어 장벽**: 영어 intent 패턴 추가 만으로 R-DWA +38%, L-DWA +49% 회복 (Option A 실증).
- **L2 도메인 어휘**: **영어 합성 대학 벤치마크** (403 QA) 를 본 논문 인프라로 구축한 결과 R-DWA F1<sub>strict</sub> **0.663** 달성 — Korean 합성의 0.529 대비 **+25%**. 방법론 자체는 **언어 중립적** 임이 실증됨 (Option B).
- **L3 아키텍처 부재**: HotpotQA 도메인-특화 재학습 trajectory 가 FLAT (Vector-only 수준) — **Triple-Hybrid 의 본질적 경계 조건** (Graph·Ontology 부재 벤치마크).
- **Option C (영어 L-DWA 학습)**: Korean-only state 에서 L-DWA ≈ R-DWA — **state 표현력이 L-DWA 가치의 상한** 임을 실증.

본 절의 실증은 초기 "naive transfer 불가" 서사를 **"원인 명확 + 해결 경로 구체"** 로 진화시켰다. 범용 일반화는 §4.2 의 다국어 intent encoder · domain-invariant state 설계 future work 의 **실증 근거** 가 된다.

## 3절 학술적·실용적 의의

### 3.1 학술적 의의

**하이브리드 RAG 연구에 강화학습 접근을 도입한 선도 사례.** 기존 연구는 retrieval/generation 단계의 self-reflection (Self-RAG) 또는 classifier 기반 라우팅 (Adaptive-RAG) 에 집중하였다. 본 논문은 **복수 지식 소스 간 연속 가중치 결정** 에 PPO 를 최초 적용하여 RAG-RL 결합의 새로운 방향을 제시한다.

**한국어 QA 평가의 실증적 기반.** F1_strict 와 F1_substring 의 0.137 vs 0.450 격차는 평가 기준에 따라 3배 이상 변동 가능함을 실증한다. 이는 한국어 NLP 평가 방법론 논의의 실증 데이터로 기여한다.

### 3.2 실용적 의의

**비용-시간 진입 장벽 완화.** 오프라인 보상 캐시로 RL 기반 RAG 튜닝의 비용을 학계·중소기업 감당 가능 수준 (일회 $33) 으로 낮추었다. 본 논문의 공개 GitHub 저장소 (https://github.com/sdw1621/triple-rag-phd) 는 `cache/university.sqlite` (16.8 MB) 와 3 seed 체크포인트를 포함하여 즉시 재현 / 재학습이 가능하다.

**도메인 특화 학습 레시피의 검증.** 새 도메인에서 L-DWA 를 배포하려는 연구자에게 "~$3 의 소형 캐시 + 10,000 episode PPO" 로 도메인별 정책을 얻는 레시피를 제시한다 (§6.7.4 참조). 이는 Graph/Ontology 가 풍부한 기업 내부 지식베이스 (법무, 의료, 금융) 에 적용 가능한 실무 경로이다.

## 4절 한계 및 향후 연구 방향

### 4.1 명시한 한계

1. **한국어 domain end-to-end 학습.** 본 논문의 L-DWA 의 "R-DWA 대비 +21.9% + Oracle 초과" 는 한국어 합성 대학에서 end-to-end 검증. 영어 합성 벤치마크 (Option B, C) 에서는 **R-DWA 가 F1<sub>strict</sub> 0.663 을 이미 달성** 하여 L-DWA 의 추가 이득은 state 표현 품질에 의해 상한이 결정됨을 실증.
2. **교차 도메인 naive transfer 실패.** §Ⅵ.7 에서 L1+L2+L3 3-층 원인 분해 및 Option A/B/C 실증 제공.
3. **EM = 0 구조적 (sentence 프롬프트 한정)**. List 프롬프트 도입으로 영어 합성 EM 0.29, 한국어 합성 EM 0.39 회복.
4. **BERT intent classifier 미학습.** 구현만 제시하고 intent_logits = (0, 0, 0) 으로 둠. Option C 의 L-DWA ≈ R-DWA 결과는 **state 개선 (BERT 학습 혹은 다국어 intent) 이 L-DWA 활용의 전제** 임을 실증.
5. **Cache 구축 시 F1 평가기 버그.** §Ⅵ.1.4 의 구두점 처리 버그를 `_PUNCT_RE` 패치로 수정, 선행 저장소에 CORRIGENDUM 으로 공개 정정 (2026-04-22).

### 4.2 향후 연구

1. **다국어 state feature.** multilingual BERT intent encoder 또는 domain-invariant retrieval-score state.
2. **도메인별 fine-tuning 레시피 체계화.** 작은 per-domain cache + 추가 PPO 의 비용-효과 분석을 HotpotQA/MuSiQue/PubMedQA 에서 수행.
3. **Reward 재설계.** EM 가중치 0.3 을 F1 (list-aware) 또는 Faithfulness 로 재분배한 버전의 실증 비교.
4. **Joint multi-domain training.** 복수 도메인 cache 를 결합하여 L-DWA 의 암묵적 도메인 clustering 검증.
5. **선행 저장소와의 평가기 통일.** `hybrid-rag-comparsion` 에 본 논문의 버그 수정을 적용하여 F1 정의를 양 프로젝트에서 통일.
6. **BERT multi-label intent 학습 후 재평가.** 특히 Conditional 유형의 추가 개선 여지.
7. **더 큰 Actor-Critic.** 현 5,636 파라미터 네트워크의 표현력 한계와 스케일링 효과 조사.

## 5절 연구 과정의 자기 성찰

본 논문의 집필 과정에서 다음 두 가지의 실측-기반 관찰이 주목할 만하였다.

**첫째, F1 평가기 버그.** 쉼표·마침표 정규화 누락으로 F1_strict 가 ~50% 과소 측정되어 있던 버그를 논문 제출 직전 (§Ⅵ.1.4) 발견하였다. 수정 후 L-DWA 의 Oracle 도달률이 52% → 99.4% 로 급격히 상승하여 본 연구의 방법론적 주장이 훨씬 강화되었다. 평가기에 대한 지속적 비판적 검토의 중요성을 실증하는 사례이다.

**둘째, 교차 도메인 transfer 실패의 재프레이밍.** 초기 계획은 "L-DWA 가 4개 벤치마크 전부에서 SOTA 를 능가" 하는 서사였으나, 실측 결과 이는 실패하였다. 본 논문은 이 실패를 **숨기지 않고** "도메인 특화 학습이 기여의 본질" 로 재프레이밍 하였다. 정직한 부정적 결과가 방법론적 주장을 오히려 강화하는 박사 학위 연구의 전형적 사례라고 판단한다.

## 6절 맺음말

본 박사 논문은 Triple-Hybrid RAG 의 가중치 결정을 학습 가능한 정책으로 전환하여, 선행 연구의 규칙 기반 접근의 구조적 한계를 극복하고 합성 대학 벤치마크에서 재현 가능한 정량 개선을 실증하였다. 오프라인 보상 캐시는 강화학습 기반 RAG 연구의 실용적 진입 장벽을 완화하며, 이중 F1 평가 체계는 한국어 QA 연구의 방법론적 투명성에 기여한다. 교차 도메인 실험은 정직한 경계 조건을 제공하며 향후 연구의 명시적 출발점이 된다.

본 연구가 제공하는 코드, 캐시, 체크포인트의 완전 공개는 후속 연구자가 본 논문의 수치를 그대로 재현하거나, 자신의 도메인에 L-DWA 를 즉시 적용하거나, 본 논문의 한계를 개선하는 연구를 시작하는 모두에 기반이 되기를 기대한다.
